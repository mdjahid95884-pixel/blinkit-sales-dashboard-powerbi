# 📊 Blinkit Sales Dashboard (Power BI)

## 🔍 Project Overview

This project is an interactive Power BI dashboard analyzing Blinkit sales performance across different outlet types, item categories, and locations.

## 📌 Key Insights

* 💰 Total Sales: $1.20M
* 📦 Total Items Sold: 9K
* ⭐ Average Rating: 3.92
* 🏪 Tier 3 outlets generated highest sales

## 📊 Dashboard Features

* KPI cards for quick business overview
* Outlet size & location analysis
* Item type performance breakdown
* Fat content-based sales analysis
* Interactive filters for dynamic exploration

## 🛠️ Tools Used

* Power BI
* DAX
* Data Modeling

## 📁 Files Included

* Blinkit_Dashboard.pbix
* dashboard.png

1. Download the `.pbix` file
